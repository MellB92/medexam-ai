# Quality Assurance Criteria - Batch Round 2

**Erstellt:** 2025-12-18
**Agent:** Opus 4.5 #1 (QA & Validation)

---

## Quality Gates (4 Kategorien)

### 1. Fachliche Korrektheit (25%)
- [ ] Medizinische Fakten korrekt
- [ ] Dosierungen/Zahlen stimmen (falls relevant)
- [ ] Keine widersprüchlichen Aussagen
- [ ] Pathophysiologie korrekt dargestellt
- [ ] Differentialdiagnosen angemessen

### 2. Leitlinien-Konformität 2024/2025 (25%)
- [ ] Aktuelle Leitlinien berücksichtigt (AWMF, S3, etc.)
- [ ] STIKO-Empfehlungen aktuell (falls Impfungen relevant)
- [ ] Keine veralteten Therapieempfehlungen
- [ ] Evidenzgrad korrekt wiedergegeben
- [ ] Referenzen/Quellen nachvollziehbar

### 3. Vollständigkeit (25%)
- [ ] Alle relevanten Aspekte behandelt
- [ ] Keine kritischen Lücken
- [ ] Kontext ausreichend erklärt
- [ ] Praktische Relevanz gegeben
- [ ] Antwort beantwortet die Frage vollständig

### 4. Sprachliche Qualität (25%)
- [ ] Verständlich formuliert
- [ ] Fachlich präzise Terminologie
- [ ] Keine Tippfehler/Grammatikfehler
- [ ] Logischer Aufbau
- [ ] Angemessene Länge

---

## Bewertungsskala

| Score | Bedeutung | Kriterien |
|-------|-----------|-----------|
| **4/4** | Perfekt | Alle 4 Kategorien erfüllt |
| **3/4** | Gut | 3 Kategorien erfüllt, 1 mit Minor Issues |
| **2/4** | Akzeptabel | 2-3 Kategorien erfüllt, größere Mängel |
| **1/4** | Mangelhaft | 1-2 Kategorien erfüllt, kritische Fehler |
| **0/4** | Unbrauchbar | Komplett falsch oder unvollständig |

---

## Pass/Fail Kriterien

**PASSED (✅):** Score >= 3/4
**MINOR ISSUES (⚠️):** Score = 2/4
**FAILED (❌):** Score <= 1/4

---

## Ziel für Batch Round 2

**Mind. 90% Pass Rate** = Mind. 9 von 10 Stichproben mit Score >= 3/4

---

## Validierungs-Prozess pro Item

1. **Frage lesen** - Verstehen was gefragt wird
2. **Original-Antwort überfliegen** - Kontext verstehen
3. **Issues prüfen** - Was waren die Probleme?
4. **Korrigierte Antwort bewerten** - Gegen Quality Gates prüfen
5. **Score vergeben** - 0-4 Punkte
6. **Notizen dokumentieren** - Begründung für Score
